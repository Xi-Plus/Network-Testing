# Network-Testing

## install

### speedtest

1. Rename `config_exmaple.php` to `config.php` and edit it.
2. Download `sql.php` at https://github.com/Xi-Plus/SQL-function
3. Download `speedtest_cli.py` at https://github.com/sivel/speedtest-cli
4. Set database.

### ping

1. Rename `config_exmaple.php` to `config.php` and edit it.
2. Download `sql.php` at https://github.com/Xi-Plus/SQL-function
